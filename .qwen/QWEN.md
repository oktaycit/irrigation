# AI Model Configuration for Irrigation Project

## Enabled Models

This project is configured to use AI models effectively for STM32 embedded development.

### Code Assistance
- **Context-aware suggestions**: STM32-specific code patterns
- **HAL Library helpers**: STM32 HAL/LL library function suggestions
- **Peripheral configuration**: GPIO, UART, SPI, I2C, ADC, TIMERS

### Code Review
- Automatic review on code changes
- Security and best practices checks
- Memory management validation

### Documentation
- Auto-generated function documentation
- Inline comment suggestions
- API documentation generation

## Usage Guidelines

1. **For code generation**: Specify the peripheral and functionality needed
2. **For debugging**: Include error messages and expected behavior
3. **For optimization**: Mention constraints (memory, speed, power)

## Project Context

- **MCU**: STM32F407VET6
- **Framework**: STM32 HAL
- **Language**: C (C99/C11)
- **IDE**: STM32CubeIDE
